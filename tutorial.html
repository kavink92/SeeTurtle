Tutorial

Properties

The pen and canvas have some properties.

Properties of pen:
x coordinate, y coordinate, angle, pencolor.

Properties of Canvas:
height, width,color.

Syntax:

The Seeturtle code has only commands and each command should be in a new line.

Commands

1. forward x:

forward x command draws a line of length x units in the direction in which it is pointing. It is not necessary that
the line should be vertical or horizontal.

2. backward x:

backward x does the same function as forward but in the oppsosite direction

3. canvascolor x y z:

canvascolor x y z assigns the color to the canvas. x,y,z are respective rgb values

4. canvassize x y :

It assigns the size to canvas. Avoid assigning size more than 600 600 as it may meddle with the other components of the page

5. pencolor x y z:

It assigns the color to the ink. x,y,z are respective rgb values

6. penwidth x :

Assigns the width (i.e) thickness of the pen

7. go x y :

go x y makes the pointer move to postion x, y without drawing a line on the canvas

8. gox x:

gox x makes the pointer move to given x coordinate without altering its y coordinate

9. goy:

goy y makes the pointer move to given y coordinate without altering its x coordinate

10. turnleft a:

Changes the angle property of the pointer (i.e) it changes the orientation. It turns the direction of pen a degrees anti-clockwise

11. turnright a:

Changes the angle property of the pointer (i.e) it changes the orientation. It turns the direction of pen a clockwise

12. direction a:

sets the angle of pen to a

13. center

Brings the pen back to the center of the canvas

14. top x/down x/left x/right x

Draws a line in corresponding direction by x units. Ex right 100 draws a line 100 units right irrespective of the pen angle

clear

clears the canvas
